# JanAushadhi Finder

## Overview

JanAushadhi Finder is an Android application developed to help users easily search for medicines, locate nearby Jan Aushadhi stores, manage medicine reminders, and view savings information. The application is designed with a simple and user-friendly interface to improve accessibility and convenience for users seeking affordable medicines.

## Objective

The main objective of this project is to provide an efficient digital platform that assists users in finding Jan Aushadhi medicines and stores while also helping them manage medicine schedules and understand cost savings.

## Features

- Medicine Search functionality
- Nearby Store Locator
- Medicine Reminder system
- Savings Information display
- Simple and responsive user interface

## Technologies Used

- Kotlin
- Jetpack Compose
- Room Database
- Android Studio

## System Description

The application is developed using Kotlin in Android Studio with Jetpack Compose for modern UI design. Room Database is used for local data storage and management. The application consists of multiple screens integrated through navigation components to provide smooth user interaction.

## Future Enhancements

- Integration with live medicine availability APIs
- User authentication and cloud synchronization
- Notification enhancements
- Dark mode support
- Online pharmacy support

## Conclusion

JanAushadhi Finder is a useful Android application that simplifies medicine search and store location services while also supporting users with medicine reminders and savings information. The project demonstrates the implementation of modern Android development concepts with a practical healthcare-related use case.
